# POS

### 프로그램 기능 
    - 멤버십 기능을 이용한 회원관리
    - 관리자 모드에서 재고관리
    - 구매 후 영수증 txt 파일로 출력
    - 카드 or 현금 결제 구분 
    
### 개발환경 
    - JAVA, Swing (GUI) , MySQL
